Academic Projects

Suspect Sketch retrieval using GAN and NLP. ML PyTorch Python

The Suspect Sketch retrieval using GAN and NLP leverages the power of Generative Adversarial Networks to create a sophisticated system capable of interpreting textual descriptions and generating corresponding images of human faces. This system is trained with a dataset comprising diverse facial images paired with descriptive texts that capture the details of each face—such as age, hair color, facial expression, and other distinguishing features. Developed a user-friendly interface where users can input text and view the generated facial images.

Network packet eavesdropping using Wireshark. NetSec Wireshark Telnet

This project involves using Wireshark, a widely-used network protocol analyzer, to capture and analyze network packets. The primary goal is to eavesdrop on network traffic in a controlled and ethical environment to gain insights into how data is transmitted and received over a network. The analysis can reveal a wealth of information about the network's operation, efficiency, and security vulnerabilities.

Network Anomaly Detection using Neural Networks. ML NetSec NNs TensorFlow Python MatplotLib

This project is centered around the development and implementation of a neural network-based system for detecting anomalies in network traffic. Anomalies in network traffic could be indicative of various security threats, such as cyber-attacks, unauthorized access attempts, or other forms of malicious activities. By using neural networks, the project aims to harness their capability for pattern recognition and learning from complex datasets to identify unusual patterns that deviate from normal network behavior.
